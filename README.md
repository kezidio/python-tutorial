# 🐍 Python Basics Programs

In this repository, I created a simple series of Python programs designed to help you learn the basics of Python while also introducing fundamental math and logic concepts such as arithmetic operations, loops, equations, string handling, and time conversion.

## 📚 Programs Included

### ➕ Addition
Demonstrates how to store values in variables and perform basic arithmetic operations. This introduces how Python handles numbers and variables in simple calculations.

### 🔁 While Loops
Shows how repetition works in programming by executing a block of code as long as a condition is true. This helps build logical thinking and introduces automation concepts.

### 📐 Quadratic Equation Solver (Roots Finder)
Builds on algebra concepts by solving equations. It calculates the discriminant to determine whether the equation has real or complex roots, helping explain when solutions exist in the real number system.

### 🌡️ Temperature Conversion
Demonstrates how mathematical formulas are applied in programming by converting Fahrenheit to Celsius, reinforcing variables and arithmetic operations.

### ⏱️ Seconds Converter
Converts a given amount of seconds into hours, minutes, and remaining seconds. This introduces division, modulus operations, and practical time-based calculations.

### 🖨️ Print Strings
Introduces how to display text output using the `print()` function. It covers printing simple messages, variables, and combining strings, helping beginners understand how programs communicate results to the user.

## 🧠 Summary

These examples are designed to be simple, clear, and beginner-friendly. Together, they build a strong foundation in Python programming while connecting coding concepts to real-world math and logic—such as how equations behave, how time is measured, and how output is displayed.
