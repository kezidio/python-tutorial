# Python Basics Programs


In this repository, I created a simple series of programs that will guide you through learning the basics of Python.
The programs included cover:

Addition – demonstrates how to store values in variables and perform basic arithmetic operations.

While Loops – shows how to repeat actions using a loop based on a condition.

Temperature Conversion – explains how to convert Fahrenheit to Celsius using a formula.

These examples are designed to be easy to understand and help build a strong foundation in Python programming.
